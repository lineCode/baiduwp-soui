# BAIDUWP
SoUI模拟的百度网盘(v5.6.3)主界面，纯界面展示，没有功能逻辑。仅用于学习SoUI使用，不要用于其它用途。

# Dependency
- [SoUI](https://github.com/SOUI2/soui.git) (git-6a12b7fe)
- Visual Studio 2015 Community
